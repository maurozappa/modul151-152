Typografie

Titel

Titel sollen gross und einfach ersichtlich sein.

Titel sind modern in einem Rot gehalten und sehen schlicht aus.

Titel haben einen grossen Abstand zu anderen Elementen und sind zentriert.

Es gibt nur einen Titel pro Seite.

Beispiel


                        #titelbeispiel { 
                            color: #dc0000;
                            font-family: 'Raleway',sans-serif;
                            font-size: 62px; 
                            font-weight: 800; 
                            line-height: 72px; 
                            margin: 0 0 24px; 
                            text-align: left; 
                            text-transform: uppercase;
                        }

Untertitel

Untertitel sind im gleichen Style wie Titel gehalten.

Jedoch sind sie kleiner und sind links anliegend, so dass sie Themen abgrenzen können.

Dieser Titel dient zum Beispiel als Artikeltitel. Es kann mehrere von diesen Titeln auf einer Seite geben.

Beispiel


                        #untertitelbeispiel { 
                            color: #dc0000;
                            font-family: 'Raleway',sans-serif;
                            font-size: 30px;
                            font-weight: 800; 
                            line-height: 36px; 
                            margin: 0 0 16px; 
                            text-align: left; 
                        }

Zwischentitel

Zwischentitel sind im gleichen Style wie andere Titel, jedoch sollen sie nicht Primär die Aufmerksamkeit auf sich ziehen.

Zwischentitel dienen dazu Textblöcke abzugrenzen.

Beispiel


                        #zwischentitel {
                            color: #dc0000; 
                            font-family:"Helvetica Neue",Arial,sans-serif; 
                            font-size: 16px; 
                            font-weight: 500; 
                            line-height: 28px; 
                            margin: 0 0 12px;
                            font-weight: bold;
                        }

Fliesstext

Fliesstext soll einfach leserlich sein und eine "passende" Grösse habe.

Er muss klein sein, um die Seite nicht zu überfüllen, jedoch nicht zu klein um schwer lesbar zu sein.

Beispiel


                        #fliesstext {
                            color: #000000; 
                            font-family:"Helvetica Neue",Arial,sans-serif; 
                            font-size: 12px; 
                            font-weight: 500; 
                            line-height: 18px; 
                        }

Bildunterschriften

Bildunterschriften sollen nicht vom Bild ablenken.

Deswegen ist diese Schrift klein, dazu wird sie durch die Schräge vom Fliesstext abgehoben.

Beispiel


                        #bildunterschriften {
                            text-align: left;
                            font-style: italic;
                            font-size: smaller;
                            text-indent: 0;
                        }

Navigationsmenu

Die Naviagion ist in der gleichen Schriftart, wie die Titel.

Jedoch hat die Navigation keine festen abstände.

Dazu ist die Farbe der Navigation dünkler als die der Titel, um sie von den Titel abzugrenzen.

Beispiel


                        #navigationsmenu {
                            color: ##706f6f; 
                            font-family:"Helvetica Neue",Arial,sans-serif; 
                            font-size: 16px; 
                            font-weight: 500;
                            font-weight: bold;
                        }

Generell

Generell gilt, die Schrift soll modern und leicht Leserlich sein.

Mit hilfe von Farben, Grössen und Schriftarten werden die Textsorten gruppiert und erleichtern somit dem User den Überblick.

Bilder

Bildsprache

Die Bilder werden einen Hauptfaktor unserer Webseite sein, daher werden diese im fokus sein und in der Mitte

Zu jedem Thema in unserem Blog wird ein Foto oder Video dabei sein.

Beispiel


                        img {
                            width: 100%;
                          }
                          
                          @media (min-width: 25em) {
                            img {
                              width: 50%;
                              float: right;
                              margin: 0 0 1em 1em;
                            }
                          }
                          
                          @media (min-width: 50em) {
                            img {
                              width: 25%;
                            }
                          }

Icons

Das Icon unserer Webseite wird zu jeder Zeit ersichtlich sein in der Navbar

Es soll jedoch nicht den fokus auf sich ziehen und eher im Hintergrund stehen
